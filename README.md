# 📥 IITC ログ保存プラグイン (Log Downloader) & 分析ツール利用ガイド

このリポジトリは、Ingress の「Intel Map」からログデータを保存し、専用ツールで詳細に分析するためのセットを提供します。

### 🌐 分析ツールの利用
保存したログの解析には、以下のホスティングサイトをご利用ください。
👉 **[Ingress Log Web (https://ingress-log.pages.dev/)](https://ingress-log.pages.dev/)**
※ `index.html` をローカルに置く必要はありません。

---

## 🚩 重要：最新バージョンへのアップデート
**2026年3月5日：大幅アップデート (Downloader v5.3.0 / Web v1.6.19)**
分析ツールの機能強化に伴い、古いバージョンのログ（JSON）では「0行表示」になるなどの不具合が発生します。必ず **Downloader を v5.3.0 以降** に更新してご利用ください。

## 🛠 手順 1：ツールをインストールする (IITC)

ブラウザの Tampermonkey アイコンからデベロッパーモードを有効にした後、以下のリンクをクリックしてインストールしてください。

### 1. 【必須】IITC 本体
[IITC 本体をインストール](https://iitc.app/build/release/total-conversion-build.user.js)

### 2. 【メイン】ログ保存ツール (Log Downloader v5.3.0)
解析ツール v1.6.19 に完全対応した最新版です。
[ログ保存ツールをインストール (v5.3.0)](https://github.com/aminosup/iitc-log-downloader/raw/main/LogDownLoader.yanwest.user.js)

---

## 🚀 手順 2：使い方

1. [Ingress Intel Map](https://intel.ingress.com/) を開きます。
2. COMM（チャット）をスクロールして、解析したい期間のログを読み込ませます。
3. 画面左下の **「📥 Log Download」** をクリックしてログを保存します。
4. **[Ingress Log Web](https://ingress-log.pages.dev/)** にアクセスし、保存したファイルをアップロードしてください。

## ✨ 更新履歴 (Changelog)

### [2026-03-05] - v1.6.19 (Web) / v5.3.0 (Downloader)
**分析精度の向上と新機能の追加**

* **📊 活動量ランキング機能**: 個別分析画面の「分析対象」をクリックすることで、ログ内のエージェントを行数順（活動量順）にリスト表示し、即座に切り替えられる機能を追加しました。
* **📍 Intel Map 連携の修正**: マップ上やリストのリンクをクリックした際、Intel Map (IITC) 側で対象ポータルが正しく自動選択（ターゲット表示）されるように URL 形式を修正しました。
* **🔴 破壊マーカーの復元**: 個別分析マップにおいて、破壊アクションのみを赤色のマーカーで強調表示する機能を復活させました。
* **📅 日時表示の適正化**: ログの表示形式を `yy/MM/DD HH:mm:ss` に統一し、視認性を向上させました。
* **🛡️ 陣営判定の修正**: レジスタンス（1/青）、エンライテンド（0/緑）の判定が逆転していた問題を完全に修正しました。
* **📁 データ構造の強化**: ツール側で正確なフィルタリングを行うために必要な `channel` および `msg` 項目を JSON に追加しました。

## ❓ うまくいかない時は？
* **ログが0行になる**: 古い Log Downloader で保存した JSON を使用していませんか？ v5.3.0 以降で保存し直してください。
* **ボタンが出ない**: Tampermonkey が有効になっているか、ページを F5 で更新したか確認してください。
* **一時保存がおかしい**: ツール画面左側の「一時保存クリア」ボタンを押してリセットしてください。
